Snapshot builds

Below are unstable releases that are automatically build by the continuous integration server. They contain all the latest and greatest features, but have the potential of eating your hamster, cat, or dog (and of course your phone). Make sure to backup before installing and using these builds. Enjoy!

Note! These snapshots now fully support Lollipop (including 'system.new.img' block image installers), but these builds are NOT compatible with ROMs patched with 8.0.0.r436.g41f104b or earlier. ROMs patched with older versions will not show up new versions of the app and vice versa.

Most people should start off fresh, but if you want to keep the currently multibooted ROMs, follow the steps below.

  1. Use the preview version of the Dual Boot Utilities and migrate the multiboot ROM data: download link
  2. Reinstall multibooted ROMs
  3. Avoid using older versions of the patcher
